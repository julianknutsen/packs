# Stripe-Level UI Polish

Do a spectacular job building absolutely world-class UI/UX components,
with an intense focus on making the most visually appealing,
user-friendly, intuitive, slick, polished, "Stripe level" of quality
UI/UX possible that leverages the good libraries already part of the
project.

## Iterative Refinement Loop

Repeat this loop at least 10 times or until diminishing returns:

1. **Evaluate** the current state of the UI component.
2. **Identify** the single biggest visual or UX improvement.
3. **Apply** the improvement.
4. **Review** the result.

## Focus Areas

- **Spacing and rhythm:** Consistent padding, margin, and whitespace.
  Elements should breathe.
- **Typography hierarchy:** Clear visual hierarchy through font size,
  weight, and color. Headers, body, captions should be immediately
  distinguishable.
- **Color harmony:** Cohesive palette. Proper contrast ratios for
  accessibility. Subtle use of color for emphasis.
- **Alignment:** Pixel-perfect alignment of elements. Grid-based layout.
- **Micro-interactions:** Hover states, transitions, focus indicators.
  Smooth, purposeful animations (not decorative).
- **Loading states:** Skeleton screens, spinners, progress indicators.
  Never leave the user wondering if something is happening.
- **Empty states:** Helpful, well-designed empty states with clear calls
  to action.
- **Error states:** Inline validation, clear error messages, recovery
  suggestions.

## Quality Bar

Reference Stripe's design system, Linear's polish, and Vercel's
dashboard for inspiration. The result should feel premium and
professional.
